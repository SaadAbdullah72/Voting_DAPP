# ⚡ MultiVote PRO - Decentralized Voting System
A professional-grade Web3 application built with **Solidity**, **Foundry**, and **React**.

## 🚀 Features
- **Multi-Election Support**: Admin can run multiple elections simultaneously.
- **Real-time Countdown**: Dynamic timers for each electoral session.
- **Gas Optimized**: Smart contract logic designed for low-cost Sepolia transactions.
- **Cyberpunk UI**: Glassmorphism design with full mobile responsiveness.

## 🛠 Tech Stack
- **Smart Contract**: Solidity 0.8.19
- **Framework**: Foundry
- **Frontend**: React + Ethers.js + Vite
- **Styling**: Bootstrap + Custom Neon CSS
